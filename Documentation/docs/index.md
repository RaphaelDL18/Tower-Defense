#Index
This is the documentation of the game **Tower Defense**. This is a personal project that I decided to do to discover the javafx library but most importantly to discover how to lead a project on my own. It is also a good opportunity to write a well-structured documentation and to improve my english skills.  
So this is a small tower defense game with 5 levels, 5 types of ennemies and 5 different towers. The goal is to defend a base against ennemies by building defensives towers along the path that lead to the said-base. Each tower needs ressources to be deployed. These ressources can be obtained by eliminating the enemies.  

This documentation is divided in 6 parts:  
- **Index**: What you're reading right now, it's the part that presents the project.  
- **Installation**: Basically how to run the game.  
- **Architecture**: How is the project's files are organized but also what are the relations between them.  
- **Usage**: How to play the game.  
- **Changelog**: List of the different updates.  

This project has been made in Java by using the javafx library.
